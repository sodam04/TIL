
# Docker 🦴

Docker = Linux Container를 만들는 컨테이너 기술이다.
```
Docker compose
컨테이너로부터 이루어진 서비스를 구축, 실행하는 순서를 자동으로 하여, 관리를 기능이다.
컴포즈를 위해서는 Docker file이 필수적으로 필요하다 
여기서 Docker file은 각자의 개발 OS에 맞춰 DB와 백엔드 서버등으로 구성되어 있다.
docker-compose.yml라는 파일 또한 필요하다
```
